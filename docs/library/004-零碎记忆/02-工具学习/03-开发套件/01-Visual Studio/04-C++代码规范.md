##### 枚举类型注释[\e]
```c++
/** @brief $SymbolContext$ */
enum AnotherEnum
{
    V1,/**< value 1 */
    V2 /**< value 2 */
};
```

##### 函数注释[\*]
```c++
/** 
* @brief: $SymbolContext$
* @param[in] 参数类型  参数名  参数说明
* @param[in|out] 参数类型  参数名  参数说明
* @return 返回值说明
* @note   
* $end$
*/
```

##### 类注释[\c]
```c++
/**
* @class $ClassName$
* @brief $SymbolContext$                                                                                         
* @note   
* $end$
*/
```

##### 文件头注释[\h]
```c++
/*****************************************************************************
*  @file     $FILE_BASE$.$FILE_EXT$
*  @brief    $SymbolContext$
*  @note
*  $end$
*
*----------------------------------------------------------------------------*
*  Change History :
* <Version> |                       <Description>
*----------------------------------------------------------------------------*
*  0.0.0.1  |                        创建头文件
*----------------------------------------------------------------------------*
*
*****************************************************************************/
```

>注意，需要安装`Visual Assist`插件哦。具体需要【VAssisX】-->【Tools】-->【Edit VA Snippets】，然后将上面的注释宏添加到打开的`VA Snippet Editor`之中
